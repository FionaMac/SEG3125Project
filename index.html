What was the motivation behind this application?

The inspiration came from the popular application myfitnesspal, which currently dominates the fitness tracking market. We felt this application was cluttered and filled with features which distracted from the main focus. We decided to create fitTrack as an alternative for users looking for a more function focused application with a clean, simple user interface. 

How did you make the application?

The frontend of the appication was created using bootstrap and features highcharts to generate the graphs. The backend was written in php and uses a mySQL database.

What is the purpose of the application?

The application is designed to be an easy way to record your daily food intake and view the caloric and macronutrient summaries by meal and by day. This allows users to tailor their eating decisions to be in line with their fitness goals, which they can adjust in their profile. Exercise tracking is coming soon.
